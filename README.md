# CRUD-PHP

 CRUD using MYSQL with PHP PDO
 
 ## Tests:
 
  - Go to the src/utils folder and configure your database options
  - Start server - php -S localhost:8000
   
 ## Observation:
 
  - You will need a user registered in the database to access the system.
